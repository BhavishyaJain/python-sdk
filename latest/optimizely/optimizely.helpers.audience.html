optimizely.helpers.audience module¶

optimizely.helpers.audience.is_match(audience, attributes)[source]¶

Given audience information and user attributes determine if user meets the conditions.

Args:
audience: Dict representing the audience. attributes: Dict representing user attributes which will be used in determining if the audience conditions are met.
Return:
Boolean representing if user satisfies audience conditions or not.
optimizely.helpers.audience.is_user_in_experiment(config, experiment, attributes)[source]¶

Determine for given experiment if user satisfies the audiences for the experiment.

Args:
config: project_config.ProjectConfig object representing the project. experiment: Object representing the experiment. attributes: Dict representing user attributes which will be used in determining if the audience conditions are met.
Returns:
Boolean representing if user satisfies audience conditions for any of the audiences or not.
