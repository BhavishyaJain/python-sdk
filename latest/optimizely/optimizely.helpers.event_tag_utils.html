optimizely.helpers.event_tag_utils module¶

optimizely.helpers.event_tag_utils.get_numeric_value(event_tags, logger=None)[source]¶

A smart getter of the numeric value from the event tags.

Args:
event_tags: A dictionary of event tags. logger: Optional logger.
Returns:

A float numeric metric value is returned when the provided numeric metric value is in the following format:

  • A string (properly formatted, e.g., no commas)
  • An integer
  • A float or double

None is returned when the provided numeric metric values is in the following format:

  • None
  • A boolean
  • inf, -inf, nan
  • A string not properly formatted (e.g., ‘1,234’)
  • Any values that cannot be cast to a float (e.g., an array or dictionary)
